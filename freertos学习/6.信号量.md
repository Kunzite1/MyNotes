# 信号量

## 6-1 简介

- 当有多个任务，同时访问或操作一个区域的资源时，会产生竞争问题，导致读取数据错误。信号量就是用来保护临界资源，协调任务、中断之间的同步性。

- 对于临界资源（同一时刻只能有一个任务或中断访问操作的资源，比如全局变量、定时器计数寄存器值），如果多个任务或中断同时调用，会产生问题。信号量通过短暂地禁止中断、挂起调度器等方法，以确保各任务或中断读取临界资源的顺序正确性。

- 信号量句柄的变量类型：`SemaphoreHandle_t`

## 6-2 几种信号量

freertos中信号量分为：二值信号量、计数型信号量、互斥信号量、递归信号量。

### 6-2-1 二值信号量

二值信号量只有0和1两个值，用来实现任务与任务、任务与中断之间的同步。可以将其视作长度为1的消息队列，因此这个消息队列只有“无消息”和“满消息”两种情况。

- 常用于任务之间的同步：当任务A访问信号量，发现为0，则任务A进入阻塞态。当任务B到达某个时刻，产生一个事件使得信号量为空，此时任务A立刻退出阻塞态，这就实现了任务A和任务B的同步。

### 6-2-2 计数型信号量

计数信号量用于事件计数和资源管理。

抽象地说，信号量就是记录当前可用资源数。当某个事件发生时，计数信号量+1。表示资源+1，当某个任务访问该资源时计数信号量-1，访问完毕后，释放该计数信号量，即计数信号量+1。

举个例子，一个有100个停车位的停车场，100个停车位相当于计数信号量初始值为100，停车场相当于资源，车就是任务。当有一辆车进入停车场，那么车位数-1，相当于计数信号量-1。当停车场停满时，即计数信号量值为0，此时没有车可以进入，除非有车出来，释放一个停车位。

### 6-2-3 互斥信号量

互斥信号量与二值信号量相似，只有空和满两种值，创建时初始为满。当有一个任务访问该临界资源时，此任务获取信号量，信号量变为空，另一个想访问此临界资源的任务B就会被阻塞，直到任务A访问结束并释放信号量，任务B才能获取信号量并访问资源。

### 6-2-4 递归信号量

没搞懂

## 6-3 常用API函数

- 要使用信号量API函数，需要包含"semphr.h"

### 6-3-1 创建二值信号量

- 使用`xSemaphoreCreateBinary()`创建二值信号量。返回值类型为`BaseType_t`，创建成功则返回信号量句柄，创建失败则返回NULL。创建时默认二值信号量为空。

- 若使用`vSemaphoreCreateBinary()`创建，则默认为满。使用该函数需要在`Freertos.h`中把宏`configSUPPORT_DYNAMIC_ALLOCATION`定义为1。

- 这两个本质上是消息队列创建函数的一个宏定义，就是一个队列长度为1，消息大小为0的一个消息队列。

### 6-3-2 创建计数信号量

- 使用`xSemaphoreCreateCounting()`创建计数信号量。使用该函数需要在`Freertos.h`中把宏`configSUPPORT_DYNAMIC_ALLOCATION`定义为1。与二值信号量相似，本质上是调用消息队列创建函数。

- 创建成功返回信号量句柄，创建失败则返回NULL。

### 6-3-3 删除信号量

`vSemaphoreDelete(SemaphoreHandle_t xSemaphore)`用于删除信号量。参数为信号量句柄。

### 6-3-4 信号量释放函数

- `xSemaphoreGive()`用于释放信号量，包括二值、计数、互斥量的释放。不可用于中断。获取成功返回`pdPASS`，获取失败则返回`errQUEUE_FULL`。

- `xSemaphoreGiveFromISR()`用于中断任务中释放信号量，与普通释放宏不同，此宏不能释放互斥量，<mark>因为互斥量不能在中断中使用</mark>。

- 这两个宏函数本质上是调用`xQueueGenericSend()`和`xQueueGiveFromISR()`，即消息队列的入队/发送。

- 释放信号量时，要注意信号量的最大值，若信号量已经为满，则无法继续调用这个宏函数。

### 6-3-5 信号量获取函数

- `xSemaphoreTake()`用于获取信号量，不可用于中断，不可获取递归互斥量。获取成功返回`pdTRUE`，获取失败返回`errQUEUE_EMPTY`。

- `xSemaphoreTakeFromISR()`用于中断任务中获取信号量。不能用于互斥信号量。

- 这两个宏函数就是调用`xQueueGenericReceive()`和`xQueueReceiveFromISR()`，就是消息队列的出队/读取。

## 6-4 示例

```c
#include "system.h"
#include "SysTick.h"
#include "led.h"
#include "usart.h"
#include "FreeRTOS.h"
#include "task.h"
#include "key.h"
#include "string.h"
#include "semphr.h"


//任务优先级
#define START_TASK_PRIO        1
//任务堆栈大小    
#define START_STK_SIZE         128  
//任务句柄
TaskHandle_t StartTask_Handler;
//任务函数
void start_task(void *pvParameters);

//任务优先级
#define LED1_TASK_PRIO        2
//任务堆栈大小    
#define LED1_STK_SIZE         50  
//任务句柄
TaskHandle_t LED1Task_Handler;
//任务函数
void led1_task(void *pvParameters);

// led2任务
#define LED2_TASK_PRIO     3
#define LED2_STK_SIZE      50
TaskHandle_t LED2Task_Handler;
void led2_task(void *pvParameters);

// 串口操作任务
#define USART_TASK_PRIO     4
#define USART_STK_SIZE      50
TaskHandle_t USARTTask_Handler;
void usart_task(void *pvParameters);

SemaphoreHandle_t BinarySem_Handler = NULL;

int main()
{
    SysTick_Init(72);
    NVIC_PriorityGroupConfig(NVIC_PriorityGroup_4);//设置系统中断优先级分组4
    LED_Init();
    USART1_Init(115200);
    printf("Program starting!\n");

    //创建开始任务
    xTaskCreate((TaskFunction_t )start_task,            //任务函数
                (const char*    )"start_task",          //任务名称
                (uint16_t       )START_STK_SIZE,        //任务堆栈大小
                (void*          )NULL,                  //传递给任务函数的参数
                (UBaseType_t    )START_TASK_PRIO,       //任务优先级
                (TaskHandle_t*  )&StartTask_Handler);   //任务句柄              
    vTaskStartScheduler();          //开启任务调度
}

//开始任务任务函数
void start_task(void *pvParameters)
{
    taskENTER_CRITICAL();           //进入临界区

    // 创建二值信号量;
    BinarySem_Handler = xSemaphoreCreateBinary();

    //创建LED1任务
    xTaskCreate((TaskFunction_t )led1_task,     
                (const char*    )"led1_task",   
                (uint16_t       )LED1_STK_SIZE, 
                (void*          )NULL,
                (UBaseType_t    )LED1_TASK_PRIO,
                (TaskHandle_t*  )&LED1Task_Handler); 
    //创建LED2任务
    xTaskCreate((TaskFunction_t )led2_task,     
                (const char*    )"led2_task",   
                (uint16_t       )LED2_STK_SIZE, 
                (void*          )NULL,
                (UBaseType_t    )LED2_TASK_PRIO,
                (TaskHandle_t*  )&LED2Task_Handler); 
    //创建USART任务
    xTaskCreate((TaskFunction_t )usart_task,     
                (const char*    )"usart_task",   
                (uint16_t       )USART_STK_SIZE, 
                (void*          )NULL,
                (UBaseType_t    )USART_TASK_PRIO,
                (TaskHandle_t*  )&USARTTask_Handler); 

    vTaskDelete(StartTask_Handler); //删除开始任务
    taskEXIT_CRITICAL();            //退出临界区
} 

//LED1任务函数
void led1_task(void *pvParameters)
{
    while(1)
    {
        LED_Ctrl(1, LIGHT_ON);
        vTaskDelay(800);
        LED_Ctrl(1, LIGHT_OFF);
        vTaskDelay(200);
    }
}

// led2任务函数
void led2_task(void *pvParameters)
{
    BaseType_t xReturn = NULL;
    while(1)
    {
        xReturn = xSemaphoreTake(BinarySem_Handler, portMAX_DELAY);
        if (xReturn == pdTRUE) {
            printf("Take BinarySem_Handler successed!\n");
            LED_Toggle(2);
        } else {
            printf("Take BinarySem_Handler failed!\n");
        }
    }
}

// 串口处理任务函数
void usart_task(void *pvParameters)
{
    BaseType_t xReturn = NULL;
    while(1)
    {
        if ((USART1_RX_STA & 0x8000) != 0) {   /*接收完成*/
            printf("%s\n", USART1_RX_BUF);

            if (USART1_RX_BUF[0] == 'a') {  /* 释放二值信号量 */
                xReturn = xSemaphoreGive(BinarySem_Handler);
            } else if (USART1_RX_BUF[0] == 'b') { /*  */

            }

            if (xReturn == pdPASS) {
                printf("Give BinarySem_Handler successed!\n");
            } else {
                printf("Give BinarySem_Handler failed!\n");
            }

            USART1_RX_STA = 0;
            memset(USART1_RX_BUF, 0, sizeof(USART1_RX_BUF));
        }
        delay_ms(10);
    }
}
```
