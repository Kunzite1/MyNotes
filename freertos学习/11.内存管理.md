# 11 内存管理

## 11-1 内存分配方法

- 当宏`configAPPLICATION_ALLOCATED_HEAP`为1时需要用户自行定义内存堆，为0时由编译器决定。自行定义时可以将内存堆定义到外部SRAM或SDRAM中。

- FreeRTOS中的内存堆为`ucHeap[]`，大小为`configTOTAL_HEAP_SIZE`（单位为字）。

- FreeRTOS提供5种内存管理方案。.c源文件存放在`FreeRTOS\Source\portable\MemMang`中

- - 对于heap_1/2/4.c这三种内存管理方案，内存堆实际上是一个很大的数组：`static uint8_t ucHeap[configTOTAL_HEAP_SIZE]`。
  - 对于heap_3.c，封装了C标准库的malloc()和free()函数，封装后具备保护，可以在嵌入式系统中使用。
  - 对于heap_5.c，每个内存堆的起始地址和大小由用户定义，可以方便的使用外部SRAM和SDRAM。

- FreeRTOS会自动对内存地址对齐，以提高性能。

---

## 11-2  heap_1.c内存分配方法

heap_1.c实现方法就是，当需要RAM的时候，从内存堆ucHeap[]中分一小块出来。

### 11-2-1 特性

- heap_1.c只能申请内存，不能释放内存，适合创建好任务、信号量、队列后就不会再删除的小应用。

- 由于无法释放内存，因此也不会产生内存碎片（一块一块碎片分布的无法使用的内存空间）。

- 代码实现很简单，内存都是从静态数组中分配的，适合不需要动态内存分配的场景。

### 11-2-2 内存申请函数pvPortMalloc()

`void *pvPortMalloc(size_t xWantedSize)`

参数：需要申请的内存大小。

返回值：申请成功的内存起始地址，申请失败则返回`NULL`。

### 11-2-3 释放函数pvFree()

`void vPortFree(void *pv)`

实际上并没有释放内存的过程。

---

## 11-3 heap_2.c分配方法

相比heap_1.c，提供内存释放函数，但不会把释放的内存块合并成一个大块，所以会产生内存碎片。

### 11-3-1 特性

- 会产生内存碎片。

- 如果一个应用动态创建和删除任务时，任务需要的堆栈大小都相同，此时heap_2.c就很适合。

- 具有不可确定性（每次执行花费时间不一致），但仍比标准C的malloc()和free()效率高。

### 11-3-2 内存堆初始化函数 prvHeapInit()

`static void prvHeapInit(void)`

初始化内存堆函数，初始化内存块、链表等。

### 11-3-3 内存申请函数 pvPortMalloc()

- 函数原型和调用方法与heap_1.c一致。

- 调用时会自动检测是否初始化内存堆。

### 11-3-3 内存释放函数 vPortFree()

- 函数原型和调用方法与heap_1.c一致。

---

## 11-4 heap_3.c内存分配方法

### 11-4-1 特性

- heap_3.c重新封装了标准C的malloc()和free()函数，使其具有保护功能。

- 很可能增大RTOS内核的代码大小。

- 具有不确定性。

- 使用heap_3.c时，FreeRtosConfig.h中的宏`configTOTAL_HEAP_SIZE`不起作用，在STM32工程中，堆由启动文件设置，单位为字节。

### 11-4-2 内存申请与释放函数

使用方法与heap_1.c和heap_2.c相同。

---

## 11-5 heap_4.c 内存分配方法

### 11-5-1 特性

- 提供一个最优的匹配算法，将内存碎片合并成一个大的可以用内存块。因此适合需要重复创建、删除任务、队列、信号量等的场景。

- 即使分配的内存大小是随机的，也不会产生内存碎片。

- 内存堆为`ucHeap[]`，大小为`configTOTAL_HEAP_SIZE`。

- 使用链表结构来管理空闲内存块，链表结构体与heap_2.c相同。

### 11-5-2 内存申请函数

- 使用方法与其他相同。

- 申请到一部分内存后，剩下的内存块组成一个新的空闲内存块，按照空闲内存块起始地址插入到空闲块链表中，合并算法会将地址相邻的空闲内存块进行合并。

### 11-5-3 内存释放函数

- 使用方法与其他相同。

- 根据释放的内存块起始地址，偏移后找到内存堆链表节点，然后将被释放的内存块插入到空闲内存块链表中，插入过程中会执行合并算法。

### 11-5-4 获取剩余内存大小

通过调用`xPortGetFreeHeapSize()`获取剩余内存大小。

---

## 11-6 heap_5.c 内存分配方法

### 11-6-1 特性

- 与heap_4.c相同的合并算法。

- 允许内存堆跨越多个不连续的内存段，比如从单片机的RAM开始，到外接的SRAM，都可以作为内存堆，其他分配方法的内存堆都不能跨越不连续的内存段。

### 11-6-2 初始化函数

`vPortDefineHeapRegions()`

- 调用API函数之前，需要先调用内存堆初始化函数`vPortDefineHeapRegions()`，<u>初始化执行完之前禁止任何可能调用`pvPortMalloc()`的操作</u>，比如创建任务、信号量、队列等等。

- 参数：`HeapRegion_t`类型的数组，用来定义内存段。

- 例如：单片机内部SRAM，起始地址为0x20000000，大小64KB。外部SRAM，起始地址0x68000000，大小1MB。则`HeapRegion_t`数组应如下定义：

```c
HeapRegion_t xHeapRegions[] =
{
    // 内部SRAM内存，起始地址0x20000000，大小64KB。
    { (uint8_t * ) 0x20000000UL, 0x10000 },
    // 起始地址0x68000000，大小1MB
    { (uint8_t * ) 0x68000000UL, 0x100000 },
}
```

## 11-7 示例

使用heap_4.c方案

```c
#include "system.h"
#include "SysTick.h"
#include "led.h"
#include "usart.h"
#include "FreeRTOS.h"
#include "task.h"
#include "key.h"
#include "string.h"
#include "time.h"

// 任务优先级
#define START_TASK_PRIO 1
// 任务堆栈大小
#define START_STK_SIZE 128
// 任务句柄
TaskHandle_t StartTask_Handler;
// 任务函数
void start_task(void *pvParameters);

// LED1任务
#define LED1_TASK_PRIO 2
#define LED1_STK_SIZE  50
TaskHandle_t LED1Task_Handler;
void led1_task(void *pvParameters);

// 串口处理任务
#define Command_TASK_PRIO 3
#define Command_STK_SIZE  512
TaskHandle_t Command_Task_handler;
void Command_task(void *pvParameters);

uint8_t *Test_Ptr = NULL; // 申请到的内存的起始地址

int main()
{
    SysTick_Init(72);
    NVIC_PriorityGroupConfig(NVIC_PriorityGroup_4); // 设置系统�?�?优先级分�?4
    LED_Init();
    USART1_Init(115200);

    printf("main: Program start\n");

    // 创建开始任务
    xTaskCreate((TaskFunction_t)start_task,          // 任务函数
                (const char *)"start_task",          // 任务名称
                (uint16_t)START_STK_SIZE,            // 任务堆栈大小
                (void *)NULL,                        // 传递给任务函数的参�?
                (UBaseType_t)START_TASK_PRIO,        // 任务优先�?
                (TaskHandle_t *)&StartTask_Handler); // 任务句柄
    vTaskStartScheduler();                           // 开启任务调度
}

// 开始任务任务函数
void start_task(void *pvParameters)
{
    taskENTER_CRITICAL(); // 进入临界�?

    // 创建LED1任务
    xTaskCreate((TaskFunction_t)led1_task,
                (const char *)"led1_task",
                (uint16_t)LED1_STK_SIZE,
                (void *)NULL,
                (UBaseType_t)LED1_TASK_PRIO,
                (TaskHandle_t *)&LED1Task_Handler);

    // 创建Command任务
    xTaskCreate((TaskFunction_t)Command_task,
                (const char *)"Command_task",
                (uint16_t)Command_STK_SIZE,
                (void *)NULL,
                (UBaseType_t)Command_TASK_PRIO,
                (TaskHandle_t *)&Command_Task_handler);

    vTaskDelete(StartTask_Handler); // 删除开始任�?
    taskEXIT_CRITICAL();            // 退出临界区
}

// LED1任务函数
void led1_task(void *pvParameters)
{
    while (1) {
        LED_Ctrl(1, LIGHT_ON);
        delay_ms(300);
        LED_Ctrl(1, LIGHT_OFF);
        delay_ms(300);
    }
}

void Command_task(void *pvParameters)
{
    uint32_t g_memsize;
    while (1) {
        if (USART1_RX_STA & 0x8000 != 0) { /* 接收完成 */
            printf("Command:%s\n", USART1_RX_BUF);

            /* 指令判断 */
            if (USART1_RX_BUF[0] == 'a') {
                if (NULL == Test_Ptr) { /* 未申请内存 */
                    // 获取剩余内存
                    g_memsize = xPortGetFreeHeapSize();
                    printf("Command_task: %d bytes of memory left\n", g_memsize);

                    Test_Ptr = pvPortMalloc(1024);

                    if (NULL != Test_Ptr) { /* 申请到内存后 */
                        printf("Command_task: Allocate memory success\n");
                        printf("Command_task: Start address: %#x\n", (int)Test_Ptr);

                        // 获取剩余内存
                        g_memsize = xPortGetFreeHeapSize();
                        printf("Command_task: %d bytes of memory left\n", g_memsize);

                        // 写入系统时间
                        sprintf((char *)Test_Ptr, "Current system TickCount = %d \n", xTaskGetTickCount());
                        printf("Command_task: Written data: \n");
                        printf("              %s\n", (char *)Test_Ptr);
                    }
                } else {
                    printf("WARNING: Free memory first\n");
                }
            }
            if (USART1_RX_BUF[0] == 'b') {
                if (NULL != Test_Ptr) {
                    printf("Command_task: Free memory\n");
                    vPortFree(Test_Ptr);
                    Test_Ptr = NULL;

                    g_memsize = xPortGetFreeHeapSize();
                    printf("Command_task: Current system memory=%d bytes,\n", g_memsize);
                    printf("              memory released success\n");
                } else {
                    printf("WARNING: Allocate memory first\n");
                }
            }
            /* END */

            // 清空串口接收
状态
            USART1_RX_STA = 0;
            memset(USART1_RX_BUF, 0, sizeof(USART1_RX_BUF));
        }
        delay_ms(20);
    }
}
```
