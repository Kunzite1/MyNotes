# 8 事件

## 8-1 简介

- 事件用于任务间通信，不能传输数据，可以一对多或者多对多。

- 任务与多个事件有两种方式建立联系：逻辑或、逻辑与。逻辑或，就是当任务感兴趣的多个任务中，有一个任务发生，即可触发；逻辑与，即任务感兴趣的多个任务中全都发生时触发。

- 事件之间是独立的，且各自与任务相关联。

- 允许多个任务对同一事件进行读写操作

- 事件变量位数取决于系统节拍计数器宏`configUSE_16_BIT_TICKS`，定义为1时，系统节拍计数器使用16位，定义为0时，32位，事件变量位数与之相同。16位时，其中8个用于存储事件组，32位时，其中24个用于存储事件组。

- 事件组就是一个变量，每一位代表一个事件，0表示事件为发生，1表示事件发生。

## 8-2 应用场景

- 常用于任务之间的同步。

- 大部分时候可以替代裸机开发的全局变量，规避了全局变量的一些问题（多任务同时访问、轮询访问全局变量会浪费CPU资源）

- 相比信号量，事件具有“逻辑或”和“逻辑与”，能实现等待多任务同步。

## 8-3 常用API函数

### 8-3-1 创建函数

`xEventGroupCreate()`：

- 使用此函数创建事件组时，采用动态内存分配。

- 创建成功则返回事件句柄，失败返回NULL，需实现定义事件句柄（变量类型`EventGroupHandle_t`）

### 8-3-2 删除函数

`vEventGroupDelete()`：

- 此函数不允许在中断中使用。

- 当事件组被删除时，所有阻塞在该事件组上的任务都会被解锁，并向等待此事件的任务返回0。

### 8-3-3 事件组置位函数

`xEventGroupSetBits()`：

- 某一位被置位后，阻塞在该位上的任务会被解锁。

- 此函数不允许在中断中使用。

`xEventGroupSetBitsFromISR()`：

- 置位函数的中断版。

### 8-3-4 等待函数

`xEventGroupWaitBits()`：

- 返回值为事件中哪些标志位被置位，可能不是用户指定的事件位，需要对返回值进行判断后再处理。

- 参数1：事件组句柄

- 参数2：感兴趣的事件，如总共8个事件，如果对bit0和bit3对应事件感兴趣，则此位为0x05(0000 1001)

- 参数3：等待到满足条件的事件后，是否将事件标志位清零。

- 参数4：pdTRUE表示对感兴趣的事件“逻辑与”，pdFALSE则“逻辑或”

- 参数5：最大超时时间。

### 8-3-5 事件标志位清除函数

`xEventGroupClearBits()`：

- 若获取事件时没有清除对应标志位，则需要使用此函数显式清除。

- 中断版本为：`xEventGroupClearBitsFromISR()`。

## 8-4 使用案例

（只展示事件发送和接收的方法，其他任务和配置不展示）

```c
#include "SysTick.h"
#include "usart.h"
#include "FreeRTOS.h"
#include "task.h"
#include "event_groups.h"
#include "string.h"

//任务优先级
#define START_TASK_PRIO        1
//任务堆栈大小    
#define START_STK_SIZE         128  
//任务句柄
TaskHandle_t StartTask_Handler;
//任务函数
void start_task(void *pvParameters);

// 事件回应任务
#define SetEvent_TASK_PRIO        3
#define SetEvent_STK_SIZE         50  
TaskHandle_t SetEvent_Task_Handler;
void SetEvent_task(void *pvParameters);

// 通讯处理任务
#define Message_TASK_PRIO   4
#define Message_STK_SIZE    50
TaskHandle_t Message_Task_Handler;
void Message_task(void *pvParameters);

// 定义事件句柄
EventGroupHandle_t Event_Handle = NULL;
// 设置事件掩码
#define EVENT1  (0x01 << 0) // bit0
#define EVENT2  (0x01 << 1) // bit1
#define EVENT3  (0x01 << 2) // bit3

int main()
{
    SysTick_Init(72);
    NVIC_PriorityGroupConfig(NVIC_PriorityGroup_4);//设置系统中断优先级分组4
    USART1_Init(115200);
    printf("Program starting!\n");

    //创建开始任务
    xTaskCreate((TaskFunction_t )start_task,            //任务函数
                (const char*    )"start_task",          //任务名称
                (uint16_t       )START_STK_SIZE,        //任务堆栈大小
                (void*          )NULL,                  //传递给任务函数的参数
                (UBaseType_t    )START_TASK_PRIO,       //任务优先级
                (TaskHandle_t*  )&StartTask_Handler);   //任务句柄              
    vTaskStartScheduler();          //开启任务调度
}

//开始任务任务函数
void start_task(void *pvParameters)
{
    taskENTER_CRITICAL();           //进入临界区

    // 创建事件
    Event_Handle = xEventGroupCreate();

    // 创建事件回应任务
    xTaskCreate((TaskFunction_t )SetEvent_task,
                (const char*    )"SetEvent_task",
                (uint16_t       )SetEvent_STK_SIZE,
                (void*          )NULL,
                (UBaseType_t    )SetEvent_TASK_PRIO,
                (TaskHandle_t   )&SetEvent_Task_Handler);
    // 创建通讯处理任务
    xTaskCreate((TaskFunction_t )Message_task,
                (const char*    )"Message_task",
                (uint16_t       )Message_STK_SIZE,
                (void*          )NULL,
                (UBaseType_t    )Message_TASK_PRIO,
                (TaskHandle_t   )&Message_Task_Handler);

    vTaskDelete(StartTask_Handler); //删除开始任务
    taskEXIT_CRITICAL();            //退出临界区
} 

// 事件回应任务函数
void SetEvent_task(void *pvParameters)
{
    EventBits_t r_event;
    while(1)
    {
        r_event = xEventGroupWaitBits(Event_Handle, 
                                        EVENT1|EVENT3|EVENT3,
                                        pdTRUE,
                                        pdFALSE,
                                        portMAX_DELAY);

        if ((r_event & EVENT3) == EVENT3) {
            printf("EVENT3 have received!\n");
        } else if ((r_event & EVENT1) == EVENT1) {
            printf("EVENT1 have received!\n");
        } else if ((r_event & EVENT2) == EVENT2) {
            printf("EVENT2 have received!\n");
        }
    }
}

// 通讯处理任务函数
void Message_task(void *pvParameters)
{
    while(1)
    {
        if ((USART1_RX_STA & 0x8000) != 0) {  /*串口接收完毕*/
            printf("Command:%s\n", USART1_RX_BUF);    // 回显接收到的指令

            // 指令判断
            if ((USART1_RX_BUF[0] == 'a') && (USART1_RX_BUF[1] == 'b')) {   /*触发事件3*/
                printf("EVENT3 triggered!\n");
                xEventGroupSetBits(Event_Handle, EVENT3);
            } else if (USART1_RX_BUF[0] == 'a') {   /*触发事件1*/
                printf("EVENT1 triggered!\n");
                xEventGroupSetBits(Event_Handle, EVENT1);
            } else if (USART1_RX_BUF[1] == 'b') {   /*触发事件2*/
                printf("EVENT2 triggered!\n");
                xEventGroupSetBits(Event_Handle, EVENT2);
            } else {
                printf("Command error!\n");
            }

            // 清空串口接收状态
            USART1_RX_STA = 0;
            memset(USART1_RX_BUF, 0, sizeof(USART1_RX_BUF));
        }
        delay_ms(10);
    }
}
```
