# USART串口通信

## 一、官方例程

### 1、路径：

\Firmware_Library\HT32_STD_5xxxx_FWLib_V1.15.2_8471\example\USART

有好几种，先看interrupt的。

### 3、代码分析

以下是USART中断例程的逐行详细解释（关键逻辑标注行号，省略重复注释部分）：

#### ①头文件与宏定义

```c
#include "ht32.h" // 芯片底层寄存器定义
#include "ht32_board.h" // 开发板硬件定义（如LED）
#include "ht32_board_config.h" // 板级配置（如USART引脚宏）
```

▶ **作用**：引入芯片支持库，定义可选的时钟校准功能

#### ②全局变量

```c
uc8 *gURTx_Ptr; // TX缓冲区指针（volatile修饰的uc8实为uint8_t）
vu32 gURTx_Length = 0; // TX剩余字节数（volatile防编译器优化）
u8 *gURRx_Ptr; // RX缓冲区指针
vu32 gURRx_Length = 0; // RX已收字节数
vu32 gIsTxFinished = FALSE; // 发送完成标志
// 发送字符串
uc8 gHelloString[] = "Hello, this is USART Tx/Rx interrupt example. Please enter 5 characters...\r\n";
u8 gTx_Buffer[128];    // 发送缓冲区数组
u8 gRx_Buffer[128];    // 接收缓冲区数组
```

▶ **关键**：`volatile`确保中断中修改的值能被主循环正确读取，避免缓存优化问题

#### ③主函数（41-70行）

```c
int main(void) {
 HT32F_DVB_LEDInit(HT_LED1); // 初始化LED1（指示发送状态）
 HT32F_DVB_LEDInit(HT_LED2); // 初始化LED2（可用于指示接收）
 gURRx_Ptr = gRx_Buffer; // RX缓冲区指针指向起始地址（[0]）
 UxART_Configuration(); // USART底层配置（核心初始化）
 UxART_TxTest(); // 启动初始发送（"Hello..."）
 while (1) UxART_RxTest(); // 主循环：处理接收数据
}
```

▶ **流程**：时钟校准→LED初始化→USART配置→先发后收→循环处理接收

#### ④USART配置函数（73-123行）

```c
void UxART_Configuration(void) {
 // 1. 使能外设时钟（AFIO、USART、GPIO）
 CKCU_PeripClockConfig_TypeDef CKCUClock = {{0}};
 CKCUClock.Bit.AFIO = 1; // 使能AFIO（引脚复用）
 CKCUClock.Bit.HTCFG_UART_RX_GPIO_CLK = 1; // 使能RX引脚所在GPIO时钟
 CKCUClock.Bit.HTCFG_UART_IPN = 1; // 使能USART模块时钟
 CKCU_PeripClockConfig(CKCUClock, ENABLE);
 // 2. 配置RX引脚内部上拉（防止悬浮干扰）
 GPIO_PullResistorConfig(HTCFG_UART_RX_GPIO_PORT, 
HTCFG_UART_RX_GPIO_PIN, GPIO_PR_UP);
 // 3. 引脚复用为USART功能（TX/RX）
 AFIO_GPxConfig(HTCFG_UART_TX_GPIO_ID, HTCFG_UART_TX_AFIO_PIN, AFIO_FUN_USART_UART);
 AFIO_GPxConfig(HTCFG_UART_RX_GPIO_ID, HTCFG_UART_RX_AFIO_PIN, AFIO_FUN_USART_UART);
 // 4. USART参数配置（115200, 8N1）
 USART_InitTypeDef USART_InitStructure = {0}; // 结构体清零（关键！）
 USART_InitStructure.USART_BaudRate = 115200; // 波特率
 USART_InitStructure.USART_WordLength = USART_WORDLENGTH_8B; // 8位数据
 USART_InitStructure.USART_StopBits = USART_STOPBITS_1; // 1位停止位
 USART_InitStructure.USART_Parity = USART_PARITY_NO; // 无校验
 USART_InitStructure.USART_Mode = USART_MODE_NORMAL; // 正常模式（非同步）
 USART_Init(HTCFG_UART_PORT, &USART_InitStructure); // 写入寄存器
 // 5. 中断配置
 NVIC_EnableIRQ(HTCFG_UART_IRQn); // 使能USART中断（NVIC层面）
 USART_IntConfig(HTCFG_UART_PORT, USART_INT_RXDR, ENABLE); // 使能RXNE中断（接收非空）
 // 6. 使能USART模块
 USART_TxCmd(HTCFG_UART_PORT, ENABLE); // 使能发送
 USART_RxCmd(HTCFG_UART_PORT, ENABLE); // 使能接收
}
```

▶ **逐行重点**：

- `{0}`初始化结构体：避免未初始化的垃圾值导致波特率计算错误

- `AFIO`配置：将普通GPIO映射为USART功能引脚

- `RXNE中断`：接收缓冲区有数据时触发（核心接收中断）

- **注意**：TX中断（TXDE/TXC）在发送时动态使能（见UxART_TxTest）

#### ⑤发送测试函数（126-136行）

```c
void UxART_TxTest(void) {
gIsTxFinished = FALSE; // 重置完成标志
gURTx_Ptr = (u8 *)gHelloString; // 指向发送字符串（"Hello..."）
gURTx_Length = sizeof(gHelloString) - 1; // 计算有效长度（排除末尾\0）
USART_IntConfig(..., USART_INT_TXDE | USART_INT_TXC, ENABLE); // 使能发送中断
while (gURTx_Length != 0); // 等待所有字节进入移位寄存器
while (gIsTxFinished == FALSE); // 等待TXC中断标志发送完成
}
```

▶ **双循环解析**：

1. `gURTx_Length`递减由TXDE中断处理，循环退出表示数据已全部写入发送寄存器

2. `gIsTxFinished`由TXC中断置1，标志整个包发送完成
   ▶ **中断触发**：
- **TXDE（发送数据寄存器空）**：每次空时触发，填充下一个字节（`*gURTx_Ptr++`）

- **TXC（传输完成）**：一帧数据发送完毕时触发，置位`gIsTxFinished`

#### ⑥接收测试函数（139-157行）

```c
void UxART_RxTest(void) {
if (gURRx_Length >= 5) { // 收到至少5个字符
// 业务处理：此处示例为回显
u32 uLength = gURRx_Length;
for (u32 i=0; i<uLength; i++) // 复制到TX缓冲区
gTx_Buffer[i] = gRx_Buffer[i];
// 回显逻辑：触发发送
gIsTxFinished = FALSE;
gURTx_Ptr = gTx_Buffer;
gURTx_Length = uLength;
USART_IntConfig(..., USART_INT_TXDE | USART_INT_TXC, ENABLE);
gURRx_Length = 0; // 重置接收计数器
}
}
```

#### ⑦USART中断服务函数

```c
void HTCFG_UART_IRQHandler(void)
{
    // 外部变量，即main.c中定义的全局变量
  extern uc8  *gURTx_Ptr;
  extern vu32  gURTx_Length;
  extern u8   *gURRx_Ptr;
  extern vu32  gURRx_Length;
  extern vu32 gIsTxFinished;
  // Rx: 将数据从UART转移到接收缓冲区
  if (USART_GetFlagStatus(HTCFG_UART_PORT, USART_FLAG_RXDR))    // 获取USART接收状态
  {
    gURRx_Ptr[gURRx_Length++] = USART_ReceiveData(HTCFG_UART_PORT);
  }

  // TX：将数据从发送缓冲区转移到UART
  if (USART_GetIntStatus(HTCFG_UART_PORT, USART_INT_TXDE) &&
      USART_GetFlagStatus(HTCFG_UART_PORT, USART_FLAG_TXDE))
  {
    if (gURTx_Length > 0)    // 还没发送完
    {
      USART_SendData(HTCFG_UART_PORT, *gURTx_Ptr++);    // 发送数据
      gURTx_Length--;    // 剩余发送字节数减一
      if (gURTx_Length == 0)    // 发送完毕
      {
        USART_IntConfig(HTCFG_UART_PORT, USART_INT_TXDE, DISABLE); // 关闭发送中断
      }
    }
  }

  if (USART_GetIntStatus(HTCFG_UART_PORT, USART_INT_TXC) &&
      USART_GetFlagStatus(HTCFG_UART_PORT, USART_FLAG_TXC))
  {
    USART_IntConfig(HTCFG_UART_PORT, USART_INT_TXC, DISABLE);
    gIsTxFinished = TRUE;
  }
}
```

#### ⑧时序分析（逻辑分析仪视角）

```
[Time 0ms] 主函数调用UxART_TxTest()
↳ [0.5ms] TX引脚开始发送"Hello..."（115200波特率，约64字节需5.5ms）
↳ [6ms] TXC中断触发，gIsTxFinished置1，主循环继续
[Time 10ms] 上位机发送5字节（如"12345"）
↳ [10.2ms] RXNE中断触发，数据存入gRx_Buffer[0]
↳ [10.3ms] RXNE中断再次触发，数据存入gRx_Buffer[1]
↳ ...（共5次中断）
↳ [10.7ms] gURRx_Length=5，主循环进入回显逻辑
↳ [11ms] 回显数据"12345"通过TX发送
```
