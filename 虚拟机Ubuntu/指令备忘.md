# WSL 2

- 在ubuntu中安装terminator，在windows中同样可以运行terminator，并且这样会直接以默认用户进入ubuntu系统。

### 基本常用指令

`wsl -u 用户名`：

- `-u`表示指定登录的用户。`-d`则是默认用户，此时将用户名换成Ubuntu22.04。

- 把用户名替换成`root`，则以root用户进入。

`wsl -l -v`：查看虚拟机状态和版本。

`wsl --shutdown`：关闭虚拟机。

`exit`：关闭当前终端。

`wsl.exe --terminate Ubuntu-22.04`：在ubuntu终端里直接关机ubuntu。

### GPU相关

`nvidia-smi`：查看显卡信息，能看到就说明GPU串口成功

# ROS

### 编译

`catkin build`：首先要进入`~/catkin_ws`工作空间

`catkin_make`：首先要进入`~/catkin_ws`工作空间

### 节点相关

`roscore`：单独启动节点前要先运行roscore。运行launch文件时则不用。

`rosrun 功能包名 节点名`：运行节点。例：`rosrun fuck_pkg haha_node`

`roslaunch 功能包名 文件名.launch`：启动launch文件。

`rosnode list`：查看活跃节点列表

`rosnode kill /节点名`：强制停止节点

`pkill -9 节点名`：强制所有名称包括“节点名”的进程。

### 创建功能包

`catkin_create_pkg 包名 依赖1 依赖2 依赖3 ..`：首先要`cd ~/catkin_ws/src`进入src目录，然后再创建功能包。例：`catkin_create_pkg fuck_pkg rospy std_msgs`

# ubuntu

`alt + shift + T`：打开终端

### 路径相关

`cd 路径`：进入某个路径

`cd ~`：回到主目录

`cd ..`：返回上一层

`ls`：查看当前路径下所有文件/文件夹

- `ls -a 路径`：显示全部文件，包括隐藏文件。

`mkdir 文件夹1 文件夹2 文件夹3`：在当前路径下创建文件夹

`rmdir 文件夹1 文件夹2 文件夹3`：在当前路径下删除文件夹

### 网络相关

`ifconfig`：显示所有网络接口状态，可用来看ip地址

`sudo systemctl restart NetworkManager`：重启网络服务。

`ip addr`：查看网卡名

`sudo dhclient ens33`：重新获取ip。`ens33`是常见的网卡名。当`ifconfig`出来没有ip地址时，可以先重启网络服务，再重新获取ip。
