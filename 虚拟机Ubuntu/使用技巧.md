# 1 扩展磁盘空间

首先关机ubuntu，然后在VMware中增加ubuntu的磁盘空间。

进入ubuntu，使用gparted工具扩展空间（先安装）：

```
sudo apt-get install gparted
sudo gparted
```

然后会出现GParted的界面：

![](../assets/2025-11-15-19-35-40-image.png)

然后右键点击/dev/sda5（是**sda5**），选择“调整大小/移动”，即可增加磁盘空间，然后点击右下角“调整大小/移动”

![](../assets/2025-11-15-19-37-47-image.png)

然后点击绿色的勾->应用->关闭，完成。

![](../assets/2025-11-15-19-41-18-image.png)
