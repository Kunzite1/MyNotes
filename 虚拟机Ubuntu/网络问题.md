# 1 右上角网络图标消失（已解决）

## 1-1 问题：

如图，这个“有线 已连接”消失的情况：

![](../assets/2025-11-15-14-48-51-image.png)

## 1-2 解决方法

ctrl+alt+T打开终端，输入命令：

```c
sudo nmcli network off
sudo nmcli network on
```

即禁用网络再开启网络，搞定。
